### Frontend-Entwicklung - die *eine Hälfte* der Webentwicklung - HTML (HyperTextMarkupLanguage) - CSS (CascadingStyleSheets) - JS (JavaScript)
### Backend-Entwicklung - die *andere Hälfte* der Webentwicklung? - Magie: für den Nutzer unsichtbar, da die Vorgänge nicht im Browser passieren - Hauptaufgabe: Validierung und Speicherung Daten für eine späteren Verwendung
### Wie kann ich eine Website erstellen?
1) Website selbst entwickeln - Programmierkenntnisse nötigt - Vorteil: freie Gestaltungsmöglichkeiten, weniger Kosten - Nachteil: Webentwicklung alleine reicht für den Erfolg der Website nicht aus (SEO, SEA, Social Media, Webdesign, UI/UX-Design, eCommerce, CMS, Webanalytics, Accessibility, Security, ...)
2) Website mit Baukastensystem bauen - Bauskastensysteme wie Jimdo, Wix, Wordpress mit Elementor, Shopify (Webshop) - Themes können verwendet werden, um vorfertigte Layouts und Designs zu verwenden - Vorteil: wenig technisches Know-How - Nachteil: generischer Look, begrenzte Möglichkeiten
2) Agenturlösung - Agentur übernimmt technischer Umsetzung - Inhalte und Design können eigenständig oder von der Agentur gestaltet werden - Vorteil: Individualität, größere Komplexität - Nachteil: Kosten
