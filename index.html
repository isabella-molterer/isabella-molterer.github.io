### Was ist eine Website? ![alt text](images/www-definition.png)
### Wie funktioniert eine Website? Eine Website liefert Inhalte an den Browser
**Dynamische** Inhalte In Form von Daten aus einer Datenbank * - Kundendaten - Produktdaten **Datenbank = Lagerhalle für verschiedene Daten und Dateien*
**Statische** Inhalte in Form von verschiedenen Dateien - Bilder: .jpg, .png, ... - Videos: .mp4, .gif, ... - Styling: .css, .scss, ... - Logik: .js

### Frontend-Entwicklung - die *eine Hälfte* der Webentwicklung - HTML (HyperTextMarkupLanguage) - CSS (CascadingStyleSheets) - JS (JavaScript)
### Backend-Entwicklung - die *andere Hälfte* der Webentwicklung - Magie: für den Nutzer unsichtbar, da die Vorgänge nicht im Browser passieren - Hauptaufgabe: Validierung und Speicherung Daten für eine späteren Verwendung
### Wie kann ich eine Website erstellen?
#### 1. Website selbst entwickeln - Programmierkenntnisse nötig - Vorteil: freie Gestaltung, weniger Kosten - Nachteil: Webentwicklung alleine reicht für den Erfolg der Website nicht aus* **SEO, SEA, Social Media, Webdesign, UI/UX-Design, eCommerce, CMS, Webanalytics, Accessibility, Security, ...*
#### 2. Website mit Baukastensystem bauen - Bauskastensysteme mit fertigen Themes - Jimdo - Wix - Wordpress mit Elementor - Shopify (Webshop) - Vorteil: wenig technisches Know-How - Nachteil: generischer Look, begrenzte Möglichkeiten
#### 3. Agenturlösung - Agentur übernimmt technischer Umsetzung - Inhalte und Design können eigenständig oder von der Agentur gestaltet werden - Vorteil: Individualität, größere Komplexität - Nachteil: Kosten
### Was sind Cookies?
- kleine Dokumente die Daten zu einer Website speichern z.B. Nutzeraktivitäten und -präferenzen - Reguläre Cookies: lokal, auf dem eigenen Gerät gespeichert - Third Party-Cookies: lokale ID, weitere Nutzerdaten auf externen Server gespeichert
